O que é o FullStackHero?

Um framework modular e opinativo para construir APIs SaaS multi-tenant e aplicações enterprise com .NET 10

Arquitetura Modular

Vertical slices com módulos independentes (Identity, Multitenancy, Auditing) que podem ser adicionados sob demanda

Pronto para Produção

Cache distribuído, jobs, mailing, storage, health checks, OpenTelemetry e muito mais já configurados

Cloud-Ready

Aspire AppHost com PostgreSQL, Redis e tracing OTLP habilitado desde o início

Quick Start

# 1. Restore dependências
dotnet restore src/FSH.Framework.slnx

# 2. Executar com Aspire (Docker necessário para Postgres/Redis)
dotnet run --project src/Playground/FSH.Playground.AppHost

# 3. Acessar a API
https://localhost:5285

# 4. Executar testes
dotnet test src/FSH.Framework.slnx
